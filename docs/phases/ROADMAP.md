## Phase 2：策略研究 + 第一个实盘（2~3 月）

**目标**：找到第一个有 Alpha 的策略，小资金实盘验证

### 策略研究路径（推荐顺序）

#### 2.1 从简单到复杂

**第一个策略推荐：双均线 + 趋势过滤**
```
原理：短期均线上穿长期均线，叠加大盘趋势过滤
优点：逻辑简单，容易实现，容易理解亏损原因
预期：年化 15%~25%，夏普 1.0~1.5
```

**第二个策略：截面动量（月频换仓）**
```
原理：每月末选取过去 3~6 月涨幅前 20% 的股票买入，持有 1 月
优点：学术上有大量支持，A 股有效
预期：年化 20%~40%，但需要处理好买卖冲击
```

**第三个策略：多因子选股（季频/月频）**
```
原理：综合价值（PB低）+ 质量（ROE高）+ 动量（近期强势）
优点：稳定性强，回撤小，适合长期运行
预期：年化 15%~30%，夏普 > 1.5
```

#### 2.2 策略研究 SOP

```
Step 1: 写出策略逻辑（文字描述）
Step 2: 实现因子计算
Step 3: IC 分析（验证因子有效性）
Step 4: 向量化回测（快速参数扫描）
Step 5: Walk-Forward 验证（防止过拟合）
Step 6: 事件驱动回测（最终验证，含完整成本）
Step 7: 纸上模拟交易（2~4 周）
Step 8: 小仓位实盘（5%~10% 资金）
Step 9: 持续监控，3 个月无异常后加仓
```
